# v0.5.0 — April 6, 2026

## New Features

### @Mentions
- **Autocomplete**: Type `@` in the composer to see a filtered list of room members. Arrow keys to navigate, Enter/Tab to confirm, Escape to dismiss.
- **Mention pills in messages**: Mentions render as accent-colored inline text. Self-mentions get a subtle background highlight. Mentions from other Matrix clients are also detected and styled.
- **Click to profile**: Hover shows pointer cursor, click opens the user's profile modal — same as clicking an avatar.
- **Composer highlighting**: `@Name` text turns accent-colored in the composer as you type.
- **Thread support**: Mention autocomplete works in both the main composer and thread panel.

### :Emoji: Shortcodes
- **Autocomplete**: Type `:` followed by 2+ characters to search emoji by shortcode (e.g., `:thumb` → 👍). Searches across 1,870 emoji using gemoji aliases and CLDR keywords.
- **Highlighted matches**: The matching portion of each shortcode is bolded in accent color.
- **Monospace display**: Shortcodes shown in JetBrains Mono with `:colons:` for a code-native feel.
- **Enter hint badge**: First/selected result shows an "Enter" badge like Slack.

### /Slash Commands
- **20 commands** across three tiers, with autocomplete on `/` at the start of a message:
  - **Common**: `/me`, `/plain`, `/join`, `/invite`, `/leave`, `/nick`, `/topic`
  - **Fun text**: `/shrug` ¯\\\_(ツ)\_/¯, `/tableflip`, `/unflip`, `/lenny`, `/spoiler`
  - **Power user**: `/op`, `/kick`, `/ban`, `/unban`, `/ignore`, `/unignore`, `/myroomnick`, `/discardsession`, `/clearcache`
- **Gloam-specific commands**: `/shrug`, `/tableflip`, `/unflip`, `/lenny` append text art. `/spoiler` wraps in `data-mx-spoiler` HTML. `/nick` and `/topic` update display name and room topic directly.
- **Error feedback**: Command failures surface as snackbars.

### State Event Messages
- **System messages in timeline**: Joins, leaves, invites, kicks, bans, room name/topic changes, encryption activation, power level changes, and join rule changes now appear as compact italic rows with contextual icons.
- **Smart collapsing**: Consecutive identical state events collapse together (e.g., "Alice, Bob, and 3 others joined the chat").
- **Non-intrusive**: State events break message grouping but use minimal vertical space. No hover toolbar, reactions, or reply capability.
- **Filtered noise**: Avatar changes are excluded (too frequent). Thread panels don't show state events.

### Notification Sounds
- **7 built-in sounds**: Soft Tap, Gentle Ping, Low Tone, Chime, Click, Pulse, Drop — synthesized tones that match Gloam's aesthetic.
- **Sound picker**: Modal with preview playback (tap play icon to hear each sound before selecting).
- **Global toggle**: Enable/disable notification sounds in Settings > Notifications.
- **Per-room overrides**: Set a different sound (or silence) for individual rooms via the room info panel.
- **Custom sounds**: Import your own `.wav`, `.mp3`, `.aiff`, or `.m4a` files via the "Add custom sound..." file picker. Custom sounds are stored in the app support directory and persist across restarts.
- **Silent option**: Mute sound for specific rooms while keeping visual notifications.
- **Test integration**: "Send test notification" in settings now plays the configured sound.

### App Icon Badge
- **macOS dock badge**: Unread DM and mention count displayed on the dock icon. Updates on every sync and clears when messages are read.
- **Windows taskbar overlay**: Numbered overlay icon (1–9, 9+) on the taskbar button for unread DMs and mentions.
- **Slack model**: Only DMs and mentions count toward the badge number. Channel-only unreads don't badge. Muted rooms are excluded entirely.
- **Auto-clear**: Badge updates immediately on app focus and when rooms are read.

### File Preview
- **Click any file attachment to preview**: opens a full-size modal (80% x 85%) with rendered content. Markdown files render with full formatting (headings, tables, blockquotes, lists). Code files render with syntax highlighting. Plain text renders in monospace. Unsupported files show metadata + download button.
- **Modal actions**: Download (⌘S), Copy All (⌘⇧C), Close (Esc) — all with keyboard shortcut tooltips.
- **Dual click targets**: clicking the file row opens preview, clicking the download icon downloads directly.
- **Encrypted room support**: modal opens immediately with loading spinner while file decrypts.
- **Selectable content**: click-and-drag text selection works across the entire preview, including code blocks.

### Markdown Upgrade
- **Full markdown rendering app-wide**: upgraded from a lightweight custom parser to `flutter_markdown`. Chat messages and file previews now support headings, tables, blockquotes, ordered/unordered lists, horizontal rules, and images.
- **Syntax-highlighted code blocks**: fenced code blocks with language tags render with full syntax highlighting in chat messages.

### Syntax Theme Picker
- **Independent code theme**: pick a syntax highlighting theme separately from the app theme. A dark code theme on a light app theme (or vice versa) is a valid combo.
- **9 curated themes**: One Dark, Dracula, Monokai, Gruvbox Dark, VS Dark, Nord, Solarized Dark, GitHub Light, Solarized Light.
- **Settings > Appearance > Code Theme**: chip selector with dark/light mode indicators.

### Appearance Preview
- **Live preview pane**: Settings > Appearance now shows a side-by-side layout — controls on the left, live preview on the right. Preview includes mock chat messages with formatting, a syntax-highlighted code block, and a room list row. Updates instantly when any setting changes.

### Connection Status
- **Connection status bar**: a slim bar below the chat header shows "Connecting...", "Reconnecting...", or "Disconnected — check your connection" based on sync state. Hidden when online.
- **Pulsing animation**: connecting and reconnecting states show a pulsing dot. Disconnected shows a solid dot.
- **15-second threshold**: brief sync errors show "Reconnecting" (SDK retries in 3s). Only sustained failures (>15s) show "Disconnected".
- **Debug controls**: Settings > Notifications has debug-only buttons (hidden in release builds) to simulate connection states for testing.

### Cross-Message Text Selection
- **Select across messages**: click and drag to select text spanning multiple messages in the chat view. Cmd+C to copy.

### Beta Release Channel
- **Update channel toggle**: Settings > About now has "update channel" (stable/beta). Beta users get pre-release builds via a separate appcast feed.
- **CI support**: tags matching `v*-beta.*` create GitHub prereleases and update only the beta appcast. Production tags update both feeds.

## Bug Fixes
- Fixed `.webm` video files being classified as audio (rendered as waveform instead of video player). Root cause: `mime` package header byte detection returned `audio/weba` for WebM containers. Fix: prioritize extension-based MIME detection.
- Fixed device name hardcoded as "Gloam (macOS)" in the user profile modal. Now reads the actual device name from the Matrix client.
- Fixed Windows notification click not navigating to the room. Root cause: Start Menu shortcut missing `AppUserModelId`, preventing Windows from routing toast activation back to the app. Fix: added AUMID to Inno Setup shortcuts + one-time startup fix for existing installs.
- Fixed Windows notifications playing the default system sound alongside the custom Gloam sound. Suppressed native toast audio with `WindowsNotificationAudio.silent()`.
- Removed non-functional "density" setting from Appearance (was only setting Flutter's `VisualDensity` which doesn't affect custom widgets).
