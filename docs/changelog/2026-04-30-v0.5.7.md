# v0.5.7 — April 30, 2026

## New Features

### Frameless Window Chrome
- **Custom titlebar across macOS, Windows, and Linux**: The standard system title bar is gone. Traffic lights (or platform window buttons on Windows/Linux) now sit on a dedicated top strip that's part of the app, not the OS chrome.
- **Centered ⌘K search pill**: A search invoker lives in the middle of the strip — clicking it (or pressing ⌘K / Ctrl+K) opens the command palette.
- **Drag and double-click-to-zoom preserved**: The whole strip is a drag region; double-click on an empty area zooms the window. Works on all three desktop platforms via `window_manager`.
- **SpaceRail nudged to 68pt**: The vertical space rail widened from 64pt to 68pt to give the macOS traffic-light cluster proper breathing room while keeping its native position.
- **Inactive-window dim**: When the window loses focus, the strip's controls dim to 60% to match macOS's desaturated traffic lights.
- **Settings modal top bar slimmed**: The settings overlay's title bar shrank to match the new strip height, dropped its label, and is now draggable end-to-end so you can move the window from inside Settings.

### Command Palette (⌘K)
- **One palette, six result groups**: Replaced the lightweight room switcher with a true command palette. ⌘K (or the strip pill) opens a centered modal with grouped results: `// channels`, `// people`, `// messages`, `// members`, `// actions`, `// themes`, `// settings`. Each section caps at 5 with a `show N more` pill that expands the section into a scrollable, keyboard-traversable list.
- **Zero-state**: Opening the palette with no query shows your top 3 most-recent rooms and top 3 most-used actions, both tracked in `SharedPreferences` and persisted across launches.
- **Local message search**: Searches the local Matrix timeline cache (last ~500 events × ~30 most-recent rooms) for body matches, debounced ~150ms. Returns snippet + room context + relative timestamp. Works offline; future work will add a richer global search.
- **Smart member jump**: Members search covers the people you actually talk to — the current room's members plus all 1:1 DM partners plus members of any joined room with ≤4 people (treating small rooms as group DMs).
- **Theme picker with live preview**: All 12 theme variants are individually selectable (type "dusk" → `Theme: dusk`). Arrowing through them previews the variant live across the whole app — including the modal backdrop, which now re-tints in real time. ↵ commits, esc reverts to whatever you had at open.
- **Action vocabulary**: `New room` · `Mark all as read` · `Mark current room as read` · `Leave current room` · `Cycle theme variant` · `Explore rooms & spaces` · `Open Settings` · `Keyboard shortcuts`. Plus deep-links into all 7 settings sections (account, appearance, voice & audio, notifications, security & keys, server, about).
- **Sidebar search bar removed**: The room-list "search or jump to" field is gone — ⌘K is now the single search/jump surface.

### Emoji Rendering
- **Inline emoji bump**: Emojis inside normal mixed text now render at 18pt (vs the 14pt body) so they're legible. Targets emoji glyphs only — body text stays the same size.
- **Jumbo emoji-only messages**: Sending a message that's nothing but emojis renders large for emphasis: 1 emoji = 44pt · 2 = 36pt · 3 = 32pt. Four or more falls back to inline. Counts unicode emoji and Matrix custom emoji (`<img data-mx-emoticon>`) toward the same threshold.
- **Reactions enlarged**: Reaction chips now use 16pt emoji (up from 13pt) for parity with the inline legibility goals.

## Bug Fixes
- **iOS notification reliability**: When `flutter_local_notifications` silently drops a delivery on iOS 26+, the AppDelegate now also registers as `UNUserNotificationCenterDelegate` and forces banner/list/badge/sound presentation. The notification diagnostic also now runs end-to-end on iOS (was macOS-only) and surfaces per-setting authorization status. Settings → Notifications gained a "request permission" tile so the system prompt can be re-triggered if the first-launch ask didn't fire.
