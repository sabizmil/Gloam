# v0.5.2 — April 7, 2026

## New Features

### Encryption Bootstrap
- **Set up encryption wizard**: a guided modal that creates SSSS keys, cross-signing keys, and enables key backup — all in one flow. Drives the Matrix SDK's `Bootstrap` state machine automatically.
- **Recovery key presentation**: displays the Base58 recovery key with both "Copy to clipboard" and "Save to file" options. Confirmation checkbox required before continuing.
- **Sidebar banner**: accent-colored prompt at the top of the room list when encryption isn't set up. "Set up encryption to secure your messages" with a "Set up" button. Dismissible, auto-hides when bootstrap completes.
- **Settings entry point**: "Set up encryption" row in Settings > Security & Keys when cross-signing isn't enabled.
