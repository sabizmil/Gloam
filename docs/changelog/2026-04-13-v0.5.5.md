# v0.5.5 — April 13, 2026

## New Features

### Attach Files Before Sending
- **Staged attachments**: Pasting, drag-and-dropping, or picking files no longer sends them immediately. Files stage in a chip strip above the composer — preview them, remove with hover-×, add up to 5 per message, and type a caption that ships alongside.
- **Click to preview**: Tap an image chip to open the fullscreen viewer before sending.
- **Captions render in the timeline**: Messages with typed captions now display the text below the image/video/file (Matrix 1.10 MSC2530).
- **Works in threads too**: Thread composer has the same staging experience, scoped per thread so drafts don't leak between them.

## Improvements

### Chat Performance
- **No more rebuild jitter**: Link previews, images, and videos no longer reload when a new message arrives. Link preview metadata is now cached on disk for 7 days with stale-while-revalidate and honors server `Cache-Control`. Image and video attachments are cached with LRU eviction. Active video playback survives timeline rebuilds — a new message won't pause your video.

### Faster Sign-in
- **Auto-detect your homeserver**: Just type your Matrix ID (`@you:server.xyz`, `you@server.xyz`, or `you:server.xyz`) and the server is extracted automatically. No more separate homeserver field for the common case.
- **Advanced toggle stays**: If your MXID suffix doesn't match your actual homeserver URL, the "set a custom server" option is still there.
- **Better error messages**: Unreachable-server errors now point at the Matrix ID and surface the custom-server option as a direct CTA.

## Bug Fixes

### Notifications
- **macOS banners no longer vanish instantly**: Notifications were defaulting to `.passive` on recent macOS, silently filing into Notification Center without showing a banner. They now show as proper banners that persist until dismissed.
- **Unread badge on the macOS dock**: Replaced an unreliable phantom-notification approach with a native dock tile API call, so the dock icon now shows your unread count (capped at `99+`).
- **Windows: clicking a notification brings the window forward**: Clicking a toast while the app is minimized or behind other windows now properly restores and focuses the window, then navigates to the room.
