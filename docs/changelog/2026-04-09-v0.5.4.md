# v0.5.4 — April 9, 2026

## New Features

### Create Space Wizard
- **Multi-step space creation flow**: new guided wizard for creating spaces with identity (name, avatar, topic), room creation, and member invites — accessible from the Explore modal's Spaces tab.
- **Delete space dialog**: admins (power level 100+) can now delete a space from the space management settings, with a confirmation dialog.

## Improvements

### CI: Linux Builds
- **Linux release artifacts**: CI now builds and publishes Linux `.tar.gz` alongside macOS and Windows on every release.

### Dev Workflow
- **Improved dev.sh**: better build script with native library bundling, code signing, and dev instance isolation.
- **DEV badge in space rail**: dev builds now show a visible "DEV" badge at the bottom of the space rail to distinguish from production.
- **Debug server expansion**: additional debug endpoints for inspecting app state during development.
- **Pre-push hook**: new git hook runs `flutter analyze` (warnings/infos non-fatal) and service tests before every push.

### Reaction State Accuracy
- **Deep reaction comparison**: timeline state change detection now compares reaction details (count, includesMe) rather than just reaction map length, preventing missed UI updates when toggling reactions.

### Explore Modal
- **Spaces tab with create action**: the Explore modal's second tab is now a dedicated Spaces tab with a "Create Space" button that launches the wizard inline.
