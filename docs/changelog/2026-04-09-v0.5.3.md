# v0.5.3 — April 9, 2026

## Improvements

### Reply Deduplication
- **Removed redundant quote blocks from replies**: replies previously showed both the Gloam reply pill and a Matrix protocol fallback blockquote below it. The `<mx-reply>` HTML fallback and plain-text `> <@user>` prefix are now stripped before rendering, leaving just the clean reply pill.

### Rich Link Previews
- **Full OG metadata in link preview cards**: link previews now display the page title, description, and hero image instead of just the bare URL and domain name.
- **Hero image layout**: OG images render full-width above the text content with a max height of 200px, respecting the image's natural aspect ratio.
- **Discord-style card design**: left accent border, bold title (2 lines max), description (3 lines max), and dimmed domain footer with external link icon.
- **Multiple URL support**: messages with multiple links now show a preview card for each URL.
- **Homeserver + client-side fallback**: tries the Matrix homeserver's `/preview_url` endpoint first. If the server doesn't support URL previews, falls back to streaming the page directly and parsing OG meta tags — stops early once metadata is found to minimize bandwidth.
- **Graceful degradation**: cards adapt to available metadata — full card with image, text-only card, or URL-only fallback when no metadata is returned.
